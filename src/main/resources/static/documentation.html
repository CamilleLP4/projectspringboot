Documentation API

Cette API permet d'interagir avec les films Star Wars.

Seul le titre, le nom du Réalisateur, la date de sortie du film et le texte d'ouverture sont utilisable.

Cette API propose 4 Endpoint :

  • Afficher le titre du Projet
  • Lister les films Star Wars
  • Afficher un film en fournissant sont ID
  • Ajouter un film à l'API
  • Activer la base de donnée

L'API fonctionne par défaut en récupérant une base de films auprès de SWAPI.

Il est possible après avoir installé la base de données (base.sql + data.sql) d'activer la base de donnée.

Afficher le titre du Projet

Retourne une chaîne de caractère correspondant au nom du projet.

Requêtes

http://localhost:8080/title

Ajouter un film à l'API

Retourne au format JSON le film fraichement crée.

Requêtes

http://localhost:8080/add?name=Nouveau Film&opening=Texte d'ouverture de ce nouveau Film&director=Le Réalisateur&date=Demain

Réponse

{ "id":10, "name":"Nouveau Film", "opening":"Texte d'ouverture de ce nouveau Film", "director":"Le Réalisateur", "date":"Demain" }

Activer la base de donnée

Active la connexion à la Base de donnée avec les paramètres fournis. Ou la désactive.

Requêtes

http://localhost:8080/db?port=3306&base=filmstarwars&user=root&password=root

http://localhost:8080/db?port=&base=api&user=&password=
